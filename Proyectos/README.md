# Catálogo de Libros en Consola


## 📘 Descripción
Este proyecto es un programa desarrollado en lenguaje C que permite gestionar un catálogo de libros desde la consola.

Permite:
- Agregar libros
- Eliminar libros por ID
- Modificar libros
- Buscar libros por título
- Mostrar todos los libros

## 🛠️ Herramientas utilizadas
- Visual Studio Code
- GCC (MinGW)
- GitHub

## ▶️ Cómo compilar

En la terminal:

gcc catalogo_libros.c -o catalogo

## ▶️ Cómo ejecutar

./catalogo

(En Windows puede ser catalogo.exe)
