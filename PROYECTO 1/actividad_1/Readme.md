# 📚 Catálogo de Libros en Consola

## 📌 Descripción
Este proyecto consiste en un programa desarrollado en lenguaje C que permite gestionar un catálogo de libros desde la consola.

El sistema permite:
- Agregar libros
- Eliminar libros por ID
- Modificar libros
- Buscar libros por título
- Mostrar todos los libros registrados

## 🛠️ Herramientas utilizadas
- Visual Studio Code
- GCC (MinGW)
- GitHub

## ⚙️ Compilación

En la terminal ejecutar:

gcc catalogo_libros.c -o catalogo_libros.exe

Luego ejecutar:

catalogo_libros.exe
